Update All Pages with Translation Support

Instructions

To add translation support to all pages in your website, follow these steps:

Option 1: Add to each page individually

Add the following code to the <head> section of each HTML page:

<script src="js/translations.js"></script>
<script src="js/global-translation.js"></script>

Option 2: Use the auto-include script (Recommended)

Add just this one line to the <head> section of each HTML page:

<script src="js/auto-include-translations.js"></script>

Option 3: Use a text editor search and replace

Use your text editor's search and replace feature to find </head> and replace it with:

<script src="js/auto-include-translations.js"></script>
</head>

Testing

After adding the script to your pages:

  1. Open any page in your browser
  2. You should see a language selector in the top-right corner
  3. Change the language and verify that text with data-translate attributes changes

Adding Translatable Text

To make text translatable, add the data-translate attribute to HTML elements:

<h1 data-translate="welcome">Welcome</h1>

Then add the translation key to the translations.js file:

en: {
  welcome: "Welcome"
},
sw: {
  welcome: "Karibu"
}
